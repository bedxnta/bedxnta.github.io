Dirty Deeds Done Dirt Cheap (D4C) is a very versatile Discord user application. It has many fun and useful features and commands that you can carry anywhere you go!


# features:

  • /avatar - Fetches the avatar of a user
  • /calc - An advanced calculator with constants and functions support
  • /define - Searches for a word in an actual dictionary or urban dictionary
  • /rcg - Generates a random color
  • /rng - Generates a random number between a provided range
  • /standinfo - Shows information about a Jojo's Bizarre Adventure Stand or Stand user
    • ... and many more are being added!
